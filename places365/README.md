# Places365 ResNet18 weights

Place `resnet18_places365.pth.tar` in this directory.

**You do NOT need to decompress it.** PyTorch loads it directly with `torch.load()`. Just copy the file:

```bash
cp /path/to/resnet18_places365.pth.tar places365/
```

Download from: http://places2.csail.mit.edu/models_places365/resnet18_places365.pth.tar
